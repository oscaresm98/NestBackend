# Teslo API

1. Clonar proyecto

2. Instalar dependencias
```
npm install
```

3. Clonar archivo __.env.template__ y renombrarlo a __.env__

4. Cambiar las variables de entorno

5. Levantar la base de datos
```
docker-compose up -d
```

6. Levantar proyecto
```
npm run start:dev
```

7. Ejecutar el seed
```
http://localhost:3000/api/seed
```

## Stack Usado
- Nest
- TypeOrm
- Postgres
